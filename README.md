# 人検出インタラクティブサウンドアプリケーション（Windows版）

USB WEBカメラの映像から人の頭部を認識し、インタラクティブに音を鳴らすアプリケーションです。2つのモードから選択できます。

> **このブランチ（`windows`）はWindows向けです。**
> コンテナは使用しません。Python仮想環境（venv）で直接実行します。

---

## 概要

このアプリケーションは、YOLOv8-poseを使用して人の頭部をリアルタイムで検出し、検出された頭部の位置に応じて音を演奏します。

### 2つのモード

起動時にモードを選択できます：

#### 1. グリッドモード（main.py）
- **16x9グリッド**: 画面を144個（16列×9行）のセルに分割
- **ピアノサウンド**: 頭部がセルに入るとDペンタトニックスケールのピアノ音が鳴る
- **ランダム音階配置**: 12秒ごとに各セルの音階がランダムに変更
- **視覚的フィードバック**: 音が鳴っているセルは半透明カラーで表示（左右反転）

#### 2. バブルモード（bubble_mode.py）
- **ふわふわ円**: 半透明のカラフルな円が画面上をランダムに移動
- **打楽器サウンド**: 頭を円に当てると打楽器音が鳴って円が消える
- **常時5個**: 画面上に常に5個の円が表示される
- **ミラーリング映像**: WEBカメラ映像を左右反転で表示

## 初回セットアップ手順

### ステップ1: Pythonのインストール確認

PowerShellを開いて、Pythonのバージョンを確認します：

```powershell
python --version
```

`Python 3.11.x` または `Python 3.12.x` と表示されればOKです。

インストールされていない場合は、[https://www.python.org/downloads/](https://www.python.org/downloads/) からダウンロードしてください。

> **インストール時の注意**: インストーラーの「**Add Python to PATH**」に必ずチェックを入れてください。

---

### ステップ2: プロジェクトの取得

```powershell
git clone https://github.com/hyousaku/park_sample.git
cd park_sample
git checkout windows
```

すでにclone済みの場合：

```powershell
cd park_sample
git checkout windows
git pull
```

---

### ステップ3: 仮想環境（venv）の作成

プロジェクトのルートディレクトリで以下を実行します：

```powershell
python -m venv venv
```

`venv` フォルダが作成されます。このフォルダにPythonの実行環境が隔離されます。

> **なぜvenvを使うのか**: システムのPythonに影響を与えず、このプロジェクト専用のライブラリバージョンを管理するためです。

---

### ステップ4: PyTorch（CPU版）のインストール

このアプリケーションはGPUがなくても動作します。CPU版のPyTorchを公式サーバーから直接インストールします。

```powershell
venv\Scripts\pip install torch==2.5.1 torchvision==0.20.1 --index-url https://download.pytorch.org/whl/cpu
```

> **なぜ別途インストールするのか**: `pip install`のデフォルトではGPU版（CUDA版）が選択される場合があり、Windows環境で正しく動作しないことがあります。CPU版を明示的に指定する必要があります。

ダウンロードサイズは約200MBです。しばらく待ちます。

---

### ステップ5: その他のライブラリのインストール

```powershell
venv\Scripts\pip install numpy==1.26.4 opencv-python==4.9.0.80 ultralytics==8.1.24 pygame==2.5.2
```

> **numpyのバージョンについて**: `numpy 2.x` 系を使うと `opencv-python 4.9.x` が正常に動作しません。必ず `1.26.4` を指定してください。

インストールが完了したら、正しくインストールされたか確認します：

```powershell
venv\Scripts\python -c "import cv2, pygame, torch, ultralytics, numpy; print('OK')"
```

`OK` と表示されればすべて正常です。

---

### ステップ6: WEBカメラの接続確認

WEBカメラをUSBポートに接続してから、認識されているか確認します：

```powershell
venv\Scripts\python -c "import cv2; [print(f'カメラ {i}: {cv2.VideoCapture(i, cv2.CAP_DSHOW).isOpened()}') for i in range(5)]"
```

`True` と表示されるインデックスが使用可能なカメラです。

---

### ステップ7: アプリケーションの起動

```powershell
venv\Scripts\python launcher.py
```

起動するとまず**カメラ選択メニュー**が表示されます：

```
==================================================
  カメラを選択してください
==================================================

カメラを検出しています...

  1. カメラ 1  (640x480)
  2. カメラ 2  (1920x1080)

==================================================
選択 (1-2):
```

使用したいカメラの番号を入力してEnterを押します。

次に**モード選択メニュー**が表示されます：

```
==================================================
  人検出インタラクティブアプリケーション
==================================================

モードを選択してください：

  1. グリッドモード（main.py）
  2. バブルモード（bubble_mode.py）
  0. 終了

==================================================
選択 (0-2):
```

**1** または **2** を入力してEnterを押してください。

> **初回起動時**: YOLOv8-poseモデル（約6.5MB）が自動的にダウンロードされます。

## アプリケーションの使用方法

### グリッドモード
1. WEBカメラの映像が全画面で表示されます
2. 頭部を動かすとYOLOが自動的に検出します
3. 頭部が画面上のグリッドセルに入ると、半透明のカラーグリッドが表示されます
4. 同時にピアノ音が鳴ります
5. 12秒ごとに各グリッドの音階が自動的に変更されます

### バブルモード
1. WEBカメラの映像が左右反転（ミラーリング）で全画面表示されます
2. カラフルな円が5個、画面上をふわふわと移動します
3. 頭を円に当てると、円が消えて打楽器音が鳴ります
4. 円が消えると新しい円が自動的に生成されます

### 終了方法
- **ESCキー**を押す
- ウィンドウの**×ボタン**をクリック

## トラブルシューティング

### カメラが一覧に表示されない

**症状**: カメラ選択画面に何も表示されない

**解決方法**:
1. WEBカメラが正しくUSBに接続されているか確認
2. Windowsのデバイスマネージャーでカメラが認識されているか確認
3. 他のアプリ（Zoom、Teamsなど）がカメラを使用中でないか確認
4. 仮想カメラソフト（iVCam、NDI Webcamなど）が干渉している場合は無効化

### `import cv2` でエラーが出る

**症状**: `numpy.core.multiarray failed to import` または `_ARRAY_API not found`

**解決方法**: numpyのバージョンが2.x系になっています。以下で1.26.4に戻してください：

```powershell
venv\Scripts\pip install numpy==1.26.4 --force-reinstall
```

### `torch` でエラーが出る

**症状**: `OSError: [WinError 193] %1 は有効な Win32 アプリケーションではありません`

**解決方法**: GPU版のtorchが混入しています。CPU版に入れ直してください：

```powershell
venv\Scripts\pip uninstall torch torchvision -y
venv\Scripts\pip install torch==2.5.1 torchvision==0.20.1 --index-url https://download.pytorch.org/whl/cpu
```

### 音が鳴らない

**症状**: 映像は表示されるが音が出ない

**解決方法**:
1. Windowsのサウンド設定でスピーカーが有効か確認
2. 他のアプリで音が出るか確認
3. pygame.mixerの初期化エラーが出ていないか確認

### `venv\Scripts\python` が認識されない

**症状**: `venv\Scripts\python` コマンドが見つからない

**解決方法**: venvが正しく作成されているか確認します：

```powershell
ls venv\Scripts\
```

`python.exe` が存在するか確認してください。存在しない場合はステップ3からやり直してください。

## 詳細な技術仕様

### 画面構成
- **解像度**: 1920×1080 (Full HD)
- **グリッド**: 16列×9行 = 144セル
- **セルサイズ**: 120×120ピクセル

### 検出システム
- **モデル**: YOLOv8n-pose（ポーズ推定用の最小モデル）
- **検出対象**: 人の頭部（鼻のキーポイントを使用）
- **検出領域**: 鼻を中心に80×80ピクセル
- **フレームレート**: 30 FPS

### サウンドシステム
- **音階**: Dペンタトニックスケール（293.66〜739.99 Hz）
- **音色**: 基音+2倍音+3倍音で合成したピアノ風サウンド
- **音の長さ**: 0.3秒（減衰エンベロープ付き）

### カメラアクセス
- **バックエンド**: DirectShow（`cv2.CAP_DSHOW`）
- **カメラ選択**: 起動時に自動検出して選択可能
- **設定方法**: `CAMERA_INDEX` 環境変数で制御
