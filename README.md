# 人検出インタラクティブサウンドアプリケーション

USB WEBカメラの映像から人の頭部を認識し、インタラクティブに音を鳴らすアプリケーションです。2つのモードから選択できます。

## 概要

このアプリケーションは、YOLOv8-poseを使用して人の頭部をリアルタイムで検出し、検出された頭部の位置に応じて音を演奏します。

### 2つのモード

起動時にモードを選択できます：

#### 1. グリッドモード（main.py）
- **16x9グリッド**: 画面を144個（16列×9行）のセルに分割
- **ピアノサウンド**: 頭部がセルに入るとDペンタトニックスケールのピアノ音が鳴る
- **ランダム音階配置**: 12秒ごとに各セルの音階がランダムに変更
- **視覚的フィードバック**: 音が鳴っているセルは半透明カラーで表示（左右反転）

#### 2. バブルモード（bubble_mode.py）
- **ふわふわ円**: 半透明のカラフルな円が画面上をランダムに移動
- **打楽器サウンド**: 頭を円に当てると打楽器音が鳴って円が消える
- **常時5個**: 画面上に常に5個の円が表示される
- **ミラーリング映像**: WEBカメラ映像を左右反転で表示

### 共通機能

- **頭部検出**: YOLOv8-poseモデルで人の頭部（鼻の位置）を正確に検出
- **Full HD映像**: 1920x1080の解像度でWEBカメラ映像を表示
- **Podman対応**: コンテナ化されており環境構築が簡単

## 必要なもの

アプリケーションを実行する前に、以下が必要です：

1. **Podman**: コンテナ実行環境
2. **USB WEBカメラ**: `/dev/video0`または`/dev/video1`として認識されるカメラ
3. **X11ディスプレイサーバー**: Linux環境（GUIが必要）

## 初心者向け起動方法

### ステップ1: Podmanのインストール確認

ターミナルを開いて、以下のコマンドを実行してください：

```bash
podman --version
```

バージョン情報が表示されればOKです。表示されない場合は、Podmanをインストールしてください：

```bash
sudo apt-get update
sudo apt-get install podman
```

### ステップ2: カメラの確認

カメラが正しく接続されているか確認します：

```bash
ls -l /dev/video*
```

`/dev/video0`や`/dev/video1`などが表示されればOKです。

### ステップ3: X11アクセスの許可

アプリケーションが画面に表示できるように、X11アクセスを許可します：

```bash
xhost +local:
```

「non-network local connections being added to access control list」と表示されればOKです。

### ステップ4: プロジェクトディレクトリに移動

```bash
cd /path/to/park_sample
```

（`/path/to/park_sample`は実際のプロジェクトの場所に置き換えてください）

### ステップ5: アプリケーションの起動

実行スクリプトに実行権限を付与して起動します：

```bash
chmod +x run.sh
./run.sh
```

初回起動時は、以下の処理が自動的に行われます：
1. Dockerイメージのビルド（数分かかります）
2. 必要なライブラリのダウンロード
3. YOLOv8-poseモデルのダウンロード（約6.5MB）
4. アプリケーションの起動

### ステップ6: モード選択

アプリケーションが起動すると、モード選択メニューが表示されます：

```
==================================================
  人検出インタラクティブアプリケーション
==================================================

モードを選択してください：

  1. グリッドモード（main.py）
     - 16x9グリッドに頭部が入るとピアノ音が鳴る
     - 12秒ごとに音階がランダムに変更

  2. バブルモード（bubble_mode.py）
     - ふわふわ移動する円に頭を当てると打楽器音が鳴る
     - 常時5個の円が画面上を移動

  0. 終了

==================================================
選択 (0-2):
```

**1**または**2**を入力してEnterキーを押してください。

### ステップ7: アプリケーションの使用

#### グリッドモードの場合：
1. **WEBカメラの映像**が全画面で表示されます
2. **頭部を動かす**と、YOLOが自動的に検出します
3. 頭部が画面上のグリッドセルに入ると、**半透明のカラーグリッド**が表示されます（左右反転）
4. 同時に**ピアノ音**が鳴ります
5. **12秒ごと**に各グリッドの音階が自動的に変更されます

#### バブルモードの場合：
1. **WEBカメラの映像**が左右反転（ミラーリング）で全画面表示されます
2. **カラフルな円**が5個、画面上をふわふわと移動します
3. **頭を円に当てる**と、円が消えて**打楽器音**が鳴ります
4. 円が消えると新しい円が自動的に生成されます
5. 常に5個の円が画面上に表示されます

### ステップ8: アプリケーションの終了

以下のいずれかの方法で終了できます：

- **ESCキー**を押す
- ウィンドウの**×ボタン**をクリック

## 詳細な技術仕様

### 画面構成

- **解像度**: 1920×1080 (Full HD)
- **グリッド**: 16列×9行 = 144セル
- **セルサイズ**: 120×120ピクセル

### 検出システム

- **モデル**: YOLOv8n-pose（ポーズ推定用の最小モデル）
- **検出対象**: 人の頭部（鼻のキーポイントを使用）
- **検出領域**: 鼻を中心に80×80ピクセル
- **フレームレート**: 30 FPS

### サウンドシステム

- **音階**: Dペンタトニックスケール
  - 周波数: 293.66, 329.63, 369.99, 440.00, 493.88, 587.33, 659.25, 739.99 Hz
- **音色**: 基音+2倍音+3倍音で合成したピアノ風サウンド
- **音の長さ**: 0.3秒（減衰エンベロープ付き）
- **音階配置**: 12秒ごとに全セルの音階がランダムに再配置

### 視覚効果

- **カラーパレット**: 6色（赤、緑、青、黄、マゼンタ、シアン）
- **透明度**: 通常100、音が鳴ると255（不透明）
- **フェードアウト**: 毎フレーム10ずつ透明度が減少
- **ミラーリング**: カラーグリッドのみ左右反転表示

## 手動実行方法（上級者向け）

`run.sh`を使わずに手動で実行する場合：

```bash
# イメージをビルド
podman build -t person-detection-app .

# コンテナを実行
podman run --rm -it \
    --name person-detection-container \
    --device=/dev/video0:/dev/video0 \
    --device=/dev/video1:/dev/video1 \
    -e DISPLAY=$DISPLAY \
    -v /tmp/.X11-unix:/tmp/.X11-unix:rw \
    --security-opt label=disable \
    --privileged \
    person-detection-app
```

## トラブルシューティング

### カメラが認識されない

**症状**: 「Error: Could not open camera」と表示される

**解決方法**:

1. カメラが接続されているか確認：
   ```bash
   ls -l /dev/video*
   ```

2. カメラのデバイス番号を確認して、`main.py`の72行目を変更：
   ```python
   self.cap = cv2.VideoCapture(0)  # 0を1や2に変更
   ```

3. カメラの権限を確認：
   ```bash
   sudo chmod 666 /dev/video0
   ```

### 画面が表示されない

**症状**: ウィンドウが開かない、または「cannot open display」エラー

**解決方法**:

1. X11アクセスを再度許可：
   ```bash
   xhost +local:
   ```

2. DISPLAYが設定されているか確認：
   ```bash
   echo $DISPLAY
   ```
   通常は`:0`や`:1`と表示されます。

3. Waylandを使用している場合、X11に切り替えるか、XWaylandを有効化してください。

### 音が鳴らない

**症状**: 映像は表示されるが音が出ない

**解決方法**:

PulseAudioを使用している場合、音声デバイスをマウント：

```bash
podman run --rm -it \
    --device=/dev/video0:/dev/video0 \
    --device=/dev/video1:/dev/video1 \
    -e DISPLAY=$DISPLAY \
    -v /tmp/.X11-unix:/tmp/.X11-unix:rw \
    -v /run/user/$(id -u)/pulse:/run/user/1000/pulse \
    -e PULSE_SERVER=unix:/run/user/1000/pulse/native \
    --security-opt label=disable \
    --privileged \
    person-detection-app
```

### アプリケーションがすぐに終了する

**症状**: 起動後すぐに終了してしまう

**解決方法**:

1. ログを確認：
   ```bash
   podman logs person-detection-container
   ```

2. カメラとディスプレイの両方が正しく動作しているか確認

3. コンテナを削除して再起動：
   ```bash
   podman rm -f person-detection-container
   ./run.sh
   ```
